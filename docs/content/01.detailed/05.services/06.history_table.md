## get_upload_by_id
ID をもとにアップロード履歴を取得する関数

#### シグネチャ
```python[history_table.py]
def get_upload_by_id(history_id: UUID) -> UploadHistory | None:
```

#### 依存するライブラリ
- **SQLAlchemy**： データベース操作を行うための ORM ライブラリ。クエリの構築や実行、トランザクション管理などを提供する。

#### 引数
| 名前       | 型   | 説明                  |
| ---------- | ---- | --------------------- |
| history_id | UUID | アップロード履歴の id |

#### 戻り値
| 型            | 説明                                                              |
| ------------- | ----------------------------------------------------------------- |
| UploadHistory | 指定された ID のアップロード履歴のレコード。存在しない場合は None |

#### 処理内容
1. `history_id` をもとにアップロード履歴のレコードをデータベースから取得する。
2. レコードが存在する場合は `UploadHistory` オブジェクトを、存在しない場合は `None` を戻り値として返す。
3.  [`SQLAlchemyError`](../02.foundation/exc.md#sqlalchemyerror) が発生した場合は [`DatabaseError`](../02.foundation/exc.md#databaseerror) を送出する。

## create_file
ファイル情報のレコードを作成する関数

#### シグネチャ
```python[history_table.py]
def create_file(file_path: str, file_content: dict, file_id: UUID | None = None) -> Files:
```

#### 依存するライブラリ
- **SQLAlchemy**： データベース操作を行うための ORM ライブラリ。クエリの構築や実行、トランザクション管理などを提供する。

#### 引数
| 名前         | 型           | 説明         |
| ------------ | ------------ | ------------ |
| file_path    | str          | ファイルパス |
| file_content | dict         | ファイル内容 |
| file_id      | UUID \| None | ファイル ID  |

#### 戻り値
| 型    | 説明                             |
| ----- | -------------------------------- |
| Files | 作成されたファイル情報のレコード |

#### 処理内容
1. `file_id` が指定されている場合はその ID を、指定されていない場合は新しい UUID をファイル ID として使用する。
2. `file_path` と `file_content` をもとにファイル情報のレコードをデータベースに作成する。
3. 作成されたファイル情報のレコードを `Files` オブジェクトを戻り値として返す。
4.  [`SQLAlchemyError`](../02.foundation/exc.md#sqlalchemyerror) が発生した場合は [`DatabaseError`](../02.foundation/exc.md#databaseerror) を送出する。

#### 依存するライブラリ
- **SQLAlchemy**： データベース操作を行うための ORM ライブラリ。クエリの構築や実行、トランザクション管理などを提供する。

#### 引数

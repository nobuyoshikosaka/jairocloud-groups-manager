## DEFAULT_LOG_FORMAT
デフォルトのログフォーマット文字列。
ログメッセージの出力形式を定義するために使用される。

#### シグネチャ
```python[const.py]
DEFAULT_LOG_FORMAT: str
```

#### 値

```python
"[%(asctime)s.%(msecs)03dZ] %(levelname)-8s %(message)s (%(addr)s - %(user)s)"
```

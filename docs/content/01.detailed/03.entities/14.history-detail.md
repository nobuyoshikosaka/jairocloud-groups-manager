## DownloadHistoryData
ダウンロード履歴を表示するためのモデルクラス

#### シグネチャ
```python[history_detail.py]
class DownloadHistoryData(pydantic.BaseModel):
```

#### 依存するライブラリ
- **Pydantic**： モデル定義および JSON シリアライズ・デシリアライズに使用。
- **uuid**： UUID 型を使用。
- **datetime**： datetime 型を使用。

#### 継承元
`pydantic.BaseModel`

#### クラス属性
| 名前             | 型                | 説明                                               |
| ---------------- | ----------------- | -------------------------------------------------- |
| id               | uuid.UUID         | 履歴データのID                                     |
| timestamp        | datetime.datetime | ダウンロード日時                                   |
| operator         | UserSummary       | ダウンロード実行ユーザー                           |
| public           | bool              | 公開・非公開設定                                   |
| parent_id        | uuid.UUID \| None | 再ダウンロード時の初回ダウンロードのID             |
| file_id          | uuid.UUID         | ファイルID                                         |
| file_path        | str               | ファイルパス                                       |
| file_exists      | bool              | ファイルの存在確認                                 |
| repository_count | int               | ダウンロードされたユーザーに関連するリポジトリの数 |
| group_count      | int               | ダウンロードされたユーザーに関連するグループの数   |
| user_count       | int               | ダウンロードされたユーザーの数                     |
| children_count   | int               | ダウンロードされたファイルの子要素の数             |

#### 注釈
- すべての属性はダウンロード履歴の表示に用いる JSON と Python オブジェクト間で camelCase ⇔ snake_case で対応している。

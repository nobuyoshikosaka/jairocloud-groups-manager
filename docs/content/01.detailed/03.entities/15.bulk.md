## RepositoryMember

```python[bulk.py]
class RepositoryMember(BaseModel):
```

#### 依存するライブラリ
- **Pydantic**： モデル定義および JSON シリアライズ・デシリアライズに使用。

#### 継承元
- `pydantic.BaseModel`

## ValidateResults
データ検証の結果を表示するためのクラス

#### シグネチャ
```python[bulk.py]
class ValidateResults(BaseModel):
```

#### 依存するライブラリ
- **Pydantic**： モデル定義および JSON シリアライズ・デシリアライズに使用。

#### 継承元
- `pydantic.BaseModel`

#### クラス属性
| 名前         | 型                               | 説明                             |
| ------------ | -------------------------------- | -------------------------------- |
| results      | list\[[EachResult](#eachresult)] | ユーザー毎の検証結果             |
| summary      | ResultSummary                    | サマリー                         |
| missing_user | list[UserDetail]                 | ファイルに含まれていないユーザー |
| offset       | int                              | オフセット                       |
| page_size    | int                              | ページサイズ                     |

#### 注釈
- すべての属性はアップロード履歴の表示に用いる JSON と Python オブジェクト間で camelCase ⇔ snake_case で対応している。
